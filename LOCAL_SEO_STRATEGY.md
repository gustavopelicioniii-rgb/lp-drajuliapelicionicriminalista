## 📍 GEO-TARGETING E SIGNALS

### Implementar no HTML:

```html
<!-- Página principal (Atibaia) -->
<meta name="geo.position" content="-23.1186;-46.5509">
<meta name="ICBM" content="-23.1186,-46.5509">
<meta name="geo.region" content="BR-SP-Atibaia">
<meta name="geo.placename" content="Atibaia, São Paulo, Brasil">

<!-- JSON-LD LocalBusiness com coordenadas exatas -->
<script type="application/ld+json">
{
  "@type": "LocalBusiness",
  "geo": {
    "@type": "GeoCoordinates",
    "latitude": "-23.1186",
    "longitude": "-46.5509"
  }
}
</script>
```
